
# chatbotAI
ChatbotAI is a project, which was created using deep learning methods to automatically answer the questions about the SAP course at university Duisburg-Essen. 
This app was written in python, css and HTML.

## Technologies
* python 3.7
* django 2.2.2
  * sklearn
  * pandas
  * numpy
  * spacy
* bootstrap 4.3.1
* jQuery 3.4.1
